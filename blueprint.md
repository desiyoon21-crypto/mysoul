# [v3] 공유 모빌리티 불법주차 관리 시스템

## 1. 개요

이 애플리케이션은 **일반 사용자**와 **두 종류의 관리자(총괄 관리자, PM 업체 관리자)**를 위한 포괄적인 불법주차 관리 플랫폼입니다. 사용자는 신고를, 관리자는 접수된 신고를 체계적으로 처리하고 관련 데이터를 관리 및 분석합니다.

---

## 2. 주요 기능 및 구현 현황

### 2.1. 사용자 역할 및 기능
*   **일반 사용자 (User):** (구현 중) 불법 주차 신고 및 상태 확인.
*   **PM 업체 관리자 (Company Admin):**
    *   **로그인:** 발급된 계정으로 로그인.
    *   **대시보드:** 담당 신고 내역 확인 (필터링).
    *   **상태 업데이트:** 처리 상태 업데이트 기능.
*   **총괄 관리자 (Super Admin):**
    *   **로그인:** 최고 관리자 계정으로 로그인.
    *   **대시보드:** 시스템 전체 현황 파악.
    *   **업체 관리:** PM 업체 CRUD (생성, 조회, 수정, 삭제).
    *   **신고 통합 관리:** 모든 신고 내역 모니터링 및 진행 상태 관리.
    *   **통계 및 리포트:** 월별 신고 건수, 처리율 등 주요 지표 확인.
    *   **엑셀 다운로드:** `xlsx` 라이브러리를 사용한 신고 내역 `.xlsx` 파일 다운로드.

### 2.2. 디자인 및 UI/UX (신규)
*   **테마 스위처 (다크/화이트 모드):**
    *   사용자 선호에 따라 다크 모드와 화이트 모드를 전환할 수 있는 기능 추가.
    *   `localStorage`를 사용하여 페이지 새로고침 후에도 테마 설정 유지.
    *   로그인 화면과 메인 애플리케이션 헤더에 테마 토글 버튼 배치.
    *   CSS 변수(Variables)를 사용한 매끄러운 테마 전환 효과.
*   **반응형 레이아웃:** 모바일 우선 디자인을 적용하여 다양한 화면 크기에서 최적화된 화면 제공.

---

## 3. 기술 스택

*   **Frontend:** HTML5, CSS3 (Modern Features), JavaScript (ES Modules).
*   **Library:** `xlsx` (Excel Export), `Font Awesome 6.4.0` (Icons).
*   **State Management:** `localStorage` (Theme Preference, Session-like data).

---

## 4. 향후 계획
