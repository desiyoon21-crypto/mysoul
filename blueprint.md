# 공유 킥보드/자전거 불법주차 신고 시스템

## 개요

이 애플리케이션은 **일반 사용자**와 **PM(Personal Mobility) 업체 관리자** 모두를 위한 모바일 웹 앱입니다. 사용자는 불법 주차된 공유 모빌리티를 신고하고 처리 상태를 추적할 수 있으며, 관리자는 접수된 신고를 확인하고 처리할 수 있습니다.

## 기능

### 일반 사용자 기능
*   **홈 화면 및 신고:** 간편한 UI를 통해 단계별로 불법 주차 기기를 신고합니다.
*   **신고 확인:** 자신의 휴대폰 번호로 신고 내역과 실시간 처리 상태(접수, 기기이동, 처리완료 등)를 목록으로 조회합니다.
*   **공지사항:** 관리자가 게시한 공지사항을 확인합니다.

### PM 업체 관리 기능
*   **관리자 페이지:** 전체 신고 내역을 목록 형태로 확인합니다.
*   **신고 상세 조회:** 각 신고 건의 상세 정보(신고 위치, 사용자 제출 사진, 위반 유형 등)를 확인합니다.
*   **처리 상태 업데이트:** 신고 건에 대해 '기기이동 완료' 등 처리 상태를 직접 업데이트합니다. 이 상태는 사용자에게도 실시간으로 공유됩니다.
*   **완료 사진 업로드:** 조치 후 현장 사진을 촬영하여 증빙 자료로 업로드합니다.

## 디자인 및 스타일

*   **통합 인터페이스:** 하단 내비게이션 바를 통해 사용자 기능과 관리자 기능을 모두 접근할 수 있도록 설계되었습니다.
*   **테마:** 일관성 있는 디자인을 유지하며, 각 기능의 목적에 맞는 명확한 UI를 제공합니다.

## 현재 작업: PM 업체 관리자 기능 구현

신고 처리 워크플로우를 위한 관리자 페이지를 구현합니다.

1.  **중앙 데이터 모델링:** 사용자의 신고와 관리자의 처리를 연동하기 위해 중앙 집중화된 신고 데이터 배열을 모델링합니다.
2.  **관리자 화면 UI 구현:**
    *   전체 신고 내역을 볼 수 있는 목록 화면을 추가합니다.
    *   신고 상세 정보와 함께 '기기이동 완료' 버튼, 완료 사진 업로드 기능을 포함한 상세 처리 화면을 구현합니다.
3.  **상태 관리 로직 구현:** 관리자가 '기기이동 완료' 버튼을 누르면, 중앙 데이터의 신고 상태가 업데이트되고, 이 변경 사항이 일반 사용자의 '신고확인' 화면에도 동적으로 반영되도록 구현합니다.
