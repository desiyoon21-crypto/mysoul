# [v4] 공유 모빌리티 불법주차 관리 시스템

## 1. 개요

이 애플리케이션은 **일반 사용자**와 **세 종류의 관리자(총괄 관리자, PM 업체 관리자, 견인 업체 관리자)**를 위한 포괄적인 불법주차 관리 플랫폼입니다. 사용자는 신고를, 관리자는 접수된 신고를 체계적으로 처리하고 관련 데이터를 관리 및 분석합니다.

---

## 2. 주요 기능 및 구현 현황

### 2.1. 사용자 역할 및 기능
*   **일반 사용자 (User):**
    *   (구현 중) 불법 주차 신고 및 상태 확인.
    *   **공지사항 확인:** 하단 메뉴의 공지사항 목록을 통해 중요 정보를 확인하고, 제목을 클릭하여 상세 내용을 아코디언 방식으로 조회.
*   **PM 업체 관리자 (Company Admin):**
    *   **로그인:** 발급된 계정으로 로그인.
    *   **대시보드:** 담당 신고 내역 확인 및 상태별(신고, 정비완료, 견인) 필터링.
    *   **상태 업데이트:** 신고 내용 상세 확인 후, 처리 사진과 내용을 첨부하여 처리 상태(기기이동, 정비완료) 변경.
*   **견인 업체 관리자 (Towing Admin):** (신규)
    *   **로그인:** 발급된 계정으로 로그인.
    *   **대시보드:** 견인 요청된 기기 목록 확인.
*   **총괄 관리자 (Super Admin):**
    *   **로그인:** 최고 관리자 계정으로 로그인.
    *   **대시보드:** 시스템 전체 현황 파악.
    *   **업체 관리:** PM 및 견인 업체 CRUD.
    *   **신고 통합 관리:** 모든 신고 내역 모니터링.
    *   **통계 및 리포트:** 월별 신고 건수, 처리율 등 주요 지표 확인.
    *   **엑셀 다운로드:** `xlsx` 라이브러리를 사용한 신고 내역 `.xlsx` 파일 다운로드.

### 2.2. 디자인 및 UI/UX
*   **테마 스위처 (다크/화이트 모드):**
    *   사용자 선호에 따라 다크 모드와 화이트 모드를 전환할 수 있는 기능.
    *   `localStorage`를 사용하여 페이지 새로고침 후에도 테마 설정 유지.
*   **반응형 레이아웃:** 모바일 우선 디자인 적용.
*   **관리자 로그인:** (신규) PM 업체, 견인업체, 관리자 탭을 포함한 새로운 로그인 화면으로 전환. 아이디/패스워드 검증 기능 포함.

---

## 3. 기술 스택

*   **Frontend:** HTML5, CSS3 (Modern Features), JavaScript (ES Modules).
*   **Library:** `xlsx` (Excel Export), `Font Awesome 6.4.0` (Icons).
*   **State Management:** `localStorage` (Theme Preference, Session-like data).

---

## 4. 현재 및 향후 계획

1.  **공지사항 UI 및 기능 구현:** `index.html`에 하단 공지사항 섹션 추가 및 `main.js`에 아코디언 기능 구현.
2.  **새로운 관리자 로그인 화면 UI 구현:** 기존 로그인 화면을 `login.html`로 분리하고, 3가지 역할(PM업체, 견인업체, 관리자)에 따른 탭 UI 구성.
3.  **관리자별 기능 구현:** 각 관리자 역할에 맞는 대시보드 및 기능 페이지 개발.
4.  백엔드 연동을 통한 데이터 영속성 확보.
