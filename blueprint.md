# [v2] 공유 모빌리티 불법주차 관리 시스템

## 1. 개요

이 애플리케이션은 **일반 사용자**와 **두 종류의 관리자(총괄 관리자, PM 업체 관리자)**를 위한 포괄적인 불법주차 관리 플랫폼입니다. 사용자는 신고를, 관리자는 접수된 신고를 체계적으로 처리하고 관련 데이터를 관리 및 분석합니다.

---

## 2. 사용자 역할 및 기능

### 2.1. 일반 사용자 (User)
*   **신고:** 웹 앱을 통해 불법 주차된 공유 모빌리티를 사진, 위치와 함께 신고합니다.
*   **상태 확인:** 자신의 휴대폰 번호로 신고한 내역과 실시간 처리 상태를 추적합니다.

### 2.2. PM 업체 관리자 (Company Admin)
*   **로그인:** 발급된 계정으로 관리자 시스템에 로그인합니다.
*   **담당 신고 확인:** 자신의 업체 소속 기기에 대한 신고 내역만 필터링하여 확인합니다.
*   **상태 업데이트:** 담당 신고 건에 대해 `정비 완료` 또는 `견인 요청` 등 처리 상태를 업데이트합니다.

### 2.3. 총괄 관리자 (Super Admin)
*   **로그인:** 최고 관리자 계정으로 시스템에 로그인합니다.
*   **대시보드:** 시스템의 핵심 기능들로 빠르게 이동할 수 있는 메인 화면입니다.
*   **홈페이지 관리:** 일반 사용자가 보는 공지사항 등 주요 콘텐츠를 관리합니다.
*   **업체 관리:** PM 업체의 정보를 등록, 수정, 삭제하고 업체별 관리자 계정을 생성합니다.
*   **신고 통합 관리:** 시스템에 접수된 **모든 신고 내역**을 실시간으로 모니터링하고 진행 상태를 관리합니다.
    *   **엑셀 다운로드:** 필터링된 신고 내역을 `.xlsx` 파일 형식으로 다운로드합니다.
*   **통계 및 리포트:** 월별/업체별 신고 건수, 처리율, 견인 요청 건수 등 주요 지표를 시각화된 차트와 표로 확인합니다.

---

## 3. 새로운 아키텍처 구현 계획

현재의 단일 페이지 구조에서 역할 기반의 다중 페이지 관리자 포털로 전환합니다.

1.  **관리자 로그인 페이지 구현:** 모든 관리자의 진입점이 될 로그인 화면을 신규 생성합니다.

2.  **총괄 관리자 포털 구축:**
    *   **대시보드 UI:** 로그인 후 보게 될 메인 대시보드 화면을 `index.html`에 추가합니다.
    *   **세부 기능 화면 추가:** `업체 관리`, `신고 통합 관리`, `통계`를 위한 별도의 HTML 섹션을 각각 구현합니다.
    *   **데이터 모델 확장:** 기존 `reports` 데이터에 `companyId`를 추가하고, `companies`와 `admins`를 관리할 새로운 데이터 구조를 `main.js`에 정의합니다.

3.  **핵심 기능 단계적 구현:**
    *   **[1단계] 업체 관리:** PM 업체 CRUD(생성, 조회, 수정, 삭제) 기능을 구현합니다.
    *   **[2단계] 엑셀 다운로드:** `npm`을 통해 `xlsx` 라이브러리를 설치하고, 신고 데이터를 엑셀 파일로 변환하여 다운로드하는 기능을 구현합니다.
    *   **[3단계] 통계 페이지:** 데이터를 집계하여 월별 리포트를 표 형태로 보여주는 기능을 구현합니다.
    *   **[4단계] PM 업체 관리자 뷰:** 업체 관리자가 로그인했을 때, 담당 신고 내역만 필터링해서 보여주는 별도의 대시보드를 구현합니다.
